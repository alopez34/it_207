Windows users:
1)Install Docker Toolbox for Windows with git bash option enabled.
2)Install Virtualbox https://www.virtualbox.org/ 
3)Place .sh script in root directory of local site folder you want mounted and hosted on LAMP server.
4)Double click script. See menu options. 

Mac Users:
1)Install Docker for mac.
2)Place .command script in root directory of local site folder you want mounted and hosted on LAMP server.
3) 
  a) run "sudo chmod a+x [filename].command
  b) Allow apps from unidentified developers for the current script. (Disable) 
4)Double click script. See menu options.
