# Install and Enable Mysql php Module

## Environment Configuration
- php 5.6.5.9-1ubuntu 4.6 cli (built: Feb 13 2015)
- Apache 2.4.7 (built: Jul 22 2014)

## Install Procedures

- apt-get udpate
